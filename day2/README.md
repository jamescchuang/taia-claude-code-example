# Day 2

## 簡介

Day 2 以 **Claude Code 進階應用** 為主軸，透過 7 個循序漸進的 Lab，帶領學員從「**設計系統建構**」、「**資料視覺化**」、「**前端改版**」、「**多代理人協作**」、「**程式碼安全檢查**」、「**MCP 整合與開發**」到「**個人知識系統建構**」，完整體驗 Claude Code 在實務開發、設計、行銷、安全、工具整合與知識管理等場景中的能力。

每個 Lab 聚焦特定 Claude Code 功能（System Prompt、Skill、Sub Agents、Hooks、Rules、MCP、CLAUDE.md 等），以實作為主、提示詞為媒介，逐步建立可重複運用的工作流。

## Lab 一覽

| Lab | 主題 | 簡介 |
| --- | --- | --- |
| [Lab 1](lab1/README.md) | 建立自己的 Claude Design | 解析 Claude Design 系統提示詞，並以提示詞與 Skill 打造自己的設計系統。 |
| [Lab 2](lab2/README.md) | 讀取 Excel 檔並建立網頁儀表板 | 讓 Claude Code 讀取 Excel 與設計規範，端到端產出本地可執行的視覺化儀表板。 |
| [Lab 3](lab3/README.md) | 向量資料庫管理與測試 | 從零生成 FastAPI + Milvus Lite 應用，並以多模態圖片輸入快速套用設計風格。 |
| [Lab 4](lab4/README.md) | Sub Agents：一人公司、多 AI 代理人 | 串接 Sub Agents、Skill、Hooks 與 CLAUDE.md，建立程式 → 行銷 → 短影音的自動化流程。 |
| [Lab 5](lab5/README.md) | Rules：在 Claude Code 執行程式碼安全檢查 | 導入 Project CodeGuard 的 Skill 與 Rules，對程式碼進行安全掃描並產出報告。 |
| [Lab 6](lab6/README.md) | MCP：使用與開發自己的 MCP Server | 從 Claude for Chrome 體驗 MCP，到開發 Reddit MCP Server 並包裝成 Skill。 |
| [Lab 7](lab7/README.md) | 建構自己的知識系統 | 依 Karpathy 的 LLM Wiki 模式，以 CLAUDE.md 為 schema 打造可持續累積的個人知識庫。 |
