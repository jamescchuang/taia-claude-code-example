## 簡介

本 Lab 以「**向量資料庫管理與測試**」應用為例，示範如何運用 Claude Code 的多模態能力與自然語言提示，加速應用程式的開發與改版：

- **從零建立專案**：以一段提示詞驅動 Claude Code 初始化基於 FastAPI + Tailwind + Milvus Lite 的向量資料庫管理應用，支援多種檔案格式索引（.txt / .md / .pdf / .py / .json / .csv / .log）。
- **召喚 Claude Code 更換頁面樣式**：將設計參考圖貼入 Claude Code（或以 `@圖檔檔名` 引用），用一句提示詞讓 Claude 自動套用設計風格並更新前端頁面。

完成本 Lab 後，學員將體驗到 Claude Code 在「**從需求到實作**」與「**從設計稿到程式碼**」兩個階段的高效協作，並理解多模態輸入與檔案參照在實務開發中的應用。

## 參考資料

### 範例畫面功能說明

示範一個使用向量資料庫的管理及測試頁面，包含下列特點：

- 使用 Ollama 來運行地端開源的 Embedding Model
- 使用者可自行管理及變更 Embedding Model
- 使用地端向量資料庫來儲存向量後的資料
- 提供使用者索引資料夾中的檔案，儲存到向量資料庫
- 提供查詢功能，回傳使用者指定的最接近資料數目
- 這個範例程式使用 Python 開發，具備網頁功能及API服務
