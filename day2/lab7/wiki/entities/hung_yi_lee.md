# 李宏毅 (Hung-yi Lee)

台灣大學電機系教授，《機器學習》與《生成式人工智慧導論》系列開放式課程的講者，本知識庫目前所有來源的作者。投影片風格以繁體中文 + 大量類比 + 漸進式視覺化見長。

## 在本知識庫中的足跡

三份「一堂課搞懂…」系列投影片，構成一條從「使用者視角 → 模型內部 → Agent 應用」的學習路徑：

1. [一堂課搞懂生成式人工智慧的原理](../sources/llm_principles.md)
2. [一堂課看懂語言模型內部運作](../sources/llm_internals.md)
3. [一堂課搞懂 Context Engineering 的概念](../sources/context_engineering.md)

更全貌的脈絡見 [李宏毅課程地圖](../synthesis/lee_curriculum_map.md)。

## 風格觀察

- 偏好用「文字接龍」、「暗無天日的小房間」、「百億參數遍地走」等口語類比解釋抽象概念。
- 強調「沒有任何模型被訓練，只訓練人類」——把焦點放在使用者可控的部分。
- 投影片含大量論文連結，可作為延伸閱讀地圖。
